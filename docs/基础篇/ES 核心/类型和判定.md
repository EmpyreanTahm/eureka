# 类型和判定

## 真值和假值

| 类型        | 换行                                         |
| :---------- | :------------------------------------------- |
| `undefined` | `false`                                      |
| `null`      | `false`                                      |
| 数          | +0、-0 和 `NaN` 是 `false`，其余是 `true`    |
| 字符串      | 长度为 0 的空字符串是 `false`，其余是 `true` |
| `Symbol`      | `true` |
| 对象        | `true`                                       |
